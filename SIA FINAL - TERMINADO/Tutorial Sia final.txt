Tutorial Sia final Gestion de Enfermeras
1. Instalar IntelliJ IDEA

2. Descargar JavaFX
   - Es necesario tener instalado JavaFX.
   - Descarga la versión correspondiente a tu sistema operativo desde la página oficial de https://openjfx.io/

3. Abrir el proyecto
   - En IntelliJ, selecciona Open.
   - Busca la carpeta donde se encuentra el archivo ProyectoSiaFinal y ábrela.

4. Configurar el JDK
   - En FILES ir a PROJECT STRUCTURE
   - En el apartado de SDK seleccionar jdk 11 nosotros utilizamos una version de Zulu 11 
     https://www.azul.com/downloads/?version=java-11-lts&architecture=x86-64-bit&package=jdk-fx&show-old-builds=true#zulu
   - En Language level seleccionar 11
   - Aplicar cambios y OK

5. Ubicar la clase principal
   - El archivo de inicio del programa es:
     src/main/java/com/example/proyectosiafinal/HelloApplication.java

6. Ejecutar la aplicación
   - Abre la clase HelloApplication.
   - Haz clic derecho sobre el archivo y selecciona Run 'HelloApplication.main()'.
   - La aplicación cargará la ventana principal del sistema y estará lista para usarse.
